Meet Cadbury, The Next Generation Search Tool

Cadbury is an Electron driven power tool to make your desktop experience even smoother. It uses modern JavaScript to simulate all sorts of results so you don't have to lift a finger. Definitions, meanings, search results, you name it! We believe with access comes power, this is why, Cadbury is an open source tool for every desktop user out there.

Simplicity Accessed

Cadbury unlike many other apps, is not window driven. The small pop up has got it all. On the click of just two buttons, the Cadbury Advanced Search is on your screen.

Cadbury Explained

This is how the app functions üëá

  • The app is launched when the shortcut keys, Ctrl + SpaceBar, are pressed
  • The value in the search box is saved to the bundle and passed on to a function
  • The function catches the value and looks up the internet for it using an API if the device is connected to the internet
  • After fetching, it displays them on your screen
  • As for other quick things, such as word definitions, its pretty straight forward just like above, it maps through a JSON file to look for the meaning and then focuses it
